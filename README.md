# java-lotto-precourse

---

## 구현 상세 목록

### 1. 입력 기능

#### 1-1. 구입 금액 입력

* 콘솔에서 사용자로부터 구입 금액 입력받음
* 1000원 단위로만 입력 가능
* 유효하지 않은 입력(문자, 음수, 0, 999 같은 단위 안 맞는 값) 시 예외 발생
* 예시

  ```
  구입금액을 입력해 주세요.
  8000
  ```

  → 로또 8장 발급

#### 1-2. 당첨 번호 입력

* 쉼표(,)로 구분된 6개의 숫자 입력받음
* 1~45 범위 내 정수여야 함
* 중복 불가
* 예시

  ```
  당첨 번호를 입력해 주세요.
  1,2,3,4,5,6
  ```

#### 1-3. 보너스 번호 입력

* 정수 1개 입력받음
* 1~45 범위 내 정수
* 당첨 번호와 중복 불가
* 예시

  ```
  보너스 번호를 입력해 주세요.
  7
  ```

---

### 2. 로또 생성 기능

#### 2-1. 로또 자동 발급

* 구입 금액 ÷ 1000 = 발급 수
* 각 로또는 1~45 범위의 숫자 6개 랜덤 생성
* 중복 불가
* 오름차순 정렬 후 리스트 형태로 저장
* 예시 출력

  ```
  8개를 구매했습니다.
  [8, 21, 23, 41, 42, 43]
  [3, 5, 11, 16, 32, 38]
  ...
  ```

---

### 3. 당첨 결과 계산

#### 3-1. 당첨 등수 기준

| 순위 | 일치 개수 | 보너스 일치 | 상금            |
| -- | ----- | ------ | ------------- |
| 1등 | 6개    | X      | 2,000,000,000 |
| 2등 | 5개    | O      | 30,000,000    |
| 3등 | 5개    | X      | 1,500,000     |
| 4등 | 4개    | X      | 50,000        |
| 5등 | 3개    | X      | 5,000         |

#### 3-2. 당첨 개수 카운트

* 각 로또별로 당첨 번호와의 일치 개수 계산
* 보너스 번호 포함 여부도 체크
* 일치 개수별로 결과 저장

#### 3-3. 출력 예시

```
당첨 통계
---
3개 일치 (5,000원) - 1개
4개 일치 (50,000원) - 0개
5개 일치 (1,500,000원) - 0개
5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
6개 일치 (2,000,000,000원) - 0개
```

---

### 4. 수익률 계산

* 총 상금 합계 / 구입 금액 * 100 = 수익률
* 소수 둘째 자리에서 반올림
* 예시 출력

  ```
  총 수익률은 62.5%입니다.
  ```

### 5. 예외 처리 요약

| 구분          | 조건                       | 예외 메시지 예시                                    |
| ----------- | ------------------------ | -------------------------------------------- |
| 구입 금액 입력 오류 | 1000원 단위 아님, 음수, 문자      | `[ERROR] 구입 금액은 1000원 단위의 양의 정수여야 합니다.`      |

| 보너스 번호 오류   | 당첨 번호 중복, 범위 벗어남         | `[ERROR] 보너스 번호는 당첨 번호와 중복되지 않아야 합니다.`       |
